# 📐 Numerical Analysis Project

A Python-based Numerical Analysis application that implements a collection of classical numerical methods used to solve mathematical and engineering problems. This project is designed for educational purposes and demonstrates the practical implementation of numerical algorithms.

---

## ✨ Features

- Root Finding Algorithms
- Numerical Differentiation
- Numerical Integration
- Matrix Operations
- Linear Equation Solvers
- Error Analysis
- User-Friendly Interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Basmalamamdouh21/Numerical-Analysis-Project.git
```

### Navigate to the project folder

```bash
cd Numerical-Analysis-Project
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
